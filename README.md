## Claude Code

- **[claude-code-tips](https://github.com/ykdojo/claude-code-tips)**
  45 tips for getting the most out of Claude Code — basics to advanced, includes custom status line script, halving the system prompt, using Gemini CLI as a subagent, and running Claude Code in a container.

- **[free-claude-code](https://github.com/Alishahryar1/free-claude-code)**
  Use Claude Code for free in the terminal, VSCode extension, or Discord (voice supported) via OpenClaw.

- **[claude-code-training](https://github.com/kousen/claude-code-training)**
  Slides and demos for a training course on Claude Code.
